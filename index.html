Tentang Game

Konsep Game

Asteroid Shooter adalah game shooter klasik di mana kamu mengendalikan pesawat luar angkasa yang harus menghancurkan asteroid yang melayang menuju ke arahmu. Tujuanmu adalah mendapatkan skor tertinggi dengan menembak asteroid sebanyak mungkin sambil menghindari tabrakan.

Cara Bermain

Gunakan tombol panah Kiri/Kanan untuk memutar pesawat. Tekan tombol panah Atas untuk maju ke arah yang dituju pesawat. Tekan Spasi untuk menembak asteroid. Jika pesawatmu bertabrakan dengan asteroid, kamu akan kehilangan nyawa. Game berakhir ketika nyawamu habis.

Teknologi

Game ini dibuat menggunakan teknologi web modern:

  • HTML5 untuk struktur halaman
  • CSS3 untuk styling dan animasi
  • JavaScript vanilla untuk logika game
  • HTML5 Canvas untuk rendering game
  • Web Audio API untuk efek suara
